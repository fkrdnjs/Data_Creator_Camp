## Overview
- 주최 측에서 제공하는 데이터 관련 3가지 과제 해결
- baseline 없이 직접 모델 구성하며 resnet 모델 사용
- wandb를 이용해 실험 과정 공유 및 tracking을 통한 과적합 판단
- 각 과제별 최종 코드, 설명, 시행착오 과정 등은 model 폴더에 정리
